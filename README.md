Description:
This is a super simple example of FastAPI, allowing quick and lazy architecture without the need to scale, demonstrating the most vital code patterns used in FastAPI applications.

1st step, required:
[Mise](https://mise.jdx.dev/)

Get started with help from Mise:
- If not prompted in terminal, run `mise trust` to trust config files
- Mise creates your virtualenv automatically
- In terminal, `mise install` adds the tools from the config to your virtualenv in .venv
- In terminal, `mise run install` adds the dependencies
- `mise run serve` serves the API
- `mise run test` runs tests
- View OpenAPI docs and tools [locally](http://127.0.0.1:8000/docs)
